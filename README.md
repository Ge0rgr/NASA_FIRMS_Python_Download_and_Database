Ein Python Skript, um die täglichen NASA FIRMS Branddaten, die als .txt abgelegt sind, herunterzuladen. Der Nutzer wählt bis zu der Tiefe der Region den Ordner mit den Daten aus, die heruntergeladen werden sollen.Dabei wird zusätzlich eine Datenbank erstellt, mittels welcher doppeltes Herunterladen vermieden wird. 
Jeden Tag wrid von NASA ein neuer Datensatz erstellt. Nur die Daten der letzten 2 Monate stehen zur Verfügung. Durch das durchführen des Skripts soll man auf einfachem Weg die aktuellsten (neuen) Daten abspeichern können.
